# VOC Intelligence — 사용자 시나리오

> 실제 데이터 기반 예시. 각 시나리오는 "누가, 언제, 왜, 어떻게" 데이터를 보는지 보여준다.

## 시나리오 1: 월요일 아침 — 사업팀 김팀장

> "내가 맡은 박두환 마스터, 이번 주 분위기 어때?"

### 경로 A: 대시보드

1. Streamlit 대시보드 접속 → **운영/사업팀 탭**
2. 마스터 헬스 스코어 한눈에 확인

```
┌─────────────────────────────────────────────────────┐
│  📊 마스터 헬스 스코어 (2026-01-05 주차)              │
├──────────┬──────┬──────┬──────┬──────────────────────┤
│ 마스터    │ 총건수│ 긍정% │ 부정% │ 전주 대비           │
├──────────┼──────┼──────┼──────┼──────────────────────┤
│ 서재형    │ 1082 │  78% │   5% │ 🟢 정상 (부정 -2%p)  │
│ 미과장    │  199 │  45% │  22% │ 🟡 주의 (부정 +8%p)  │
│ 박두환    │  565 │  52% │  28% │ 🔴 위험 (부정 +13%p) │ ← ⚠️
│ 홍춘욱    │   20 │  60% │  15% │ 🟢 정상              │
│ ...       │      │      │      │                      │
└──────────┴──────┴──────┴──────┴──────────────────────┘
```

3. 박두환 🔴 클릭 → 드릴다운

```
┌─────────────────────────────────────────────────────┐
│  박두환 상세 (2026-01-05 주차)                        │
│  편지 565건 (전주 410, +155) / 게시글 0건              │
├─────────────────────────────────────────────────────┤
│                                                      │
│  ■ 부정 의견 분석 (158건, 28%)                         │
│                                                      │
│  category_tags 분포:                                  │
│  ┌──────────────────────────────────┬─────┐          │
│  │ 마스터 소통/태도                    │  82 │ ████████ │
│  │ 수익/손실 공유                     │  35 │ ████     │
│  │ 커뮤니티 분위기                    │  24 │ ███      │
│  │ 투자 학습 질문                     │  17 │ ██       │
│  └──────────────────────────────────┴─────┘          │
│                                                      │
│  ■ 핵심 주제 클러스터                                  │
│                                                      │
│  🔥 수익인증 요구 논란 (~80건)                         │
│  ├─ topic: 투자 담론 / sentiment: 부정                 │
│  ├─ free_tags: "수익인증", "투명성", "체결내역서"        │
│  └─ 대표 원문: _"증권사 공식 주문 체결 내역서나          │
│     소득금액증명원을 단 한 번이라도 투명하게 공개해       │
│     주십시오..."_                                     │
│                                                      │
│  💬 마스터 옹호 반응 (~150건)                           │
│  ├─ topic: 커뮤니티 소통 / sentiment: 긍정              │
│  ├─ free_tags: "투자철학", "신뢰", "옹호"               │
│  └─ 대표 원문: _"두환쌤의 수익인증이 왜 필요합니까?      │
│     인신 공격으로 비춰집니다..."_                       │
│                                                      │
```

4. "4주 추이 보기" 클릭

### 경로 B: Slack Bot

```
김팀장: /voc 박두환 이번주 요약

VOC Bot:
  📋 박두환 (2026-01-05 주차)
  ──────────────────────
  총 565건 (전주 대비 +155건, +38%)

  ⚠️ 부정 비율 28% (전주 15%, +13%p)

  핵심 이슈:
  1. 수익인증 요구 논란 (~80건)
     → 일부 이용자가 증권사 체결내역 공개를 요구
  2. 마스터 옹호 반응 (~150건)
     → 다수가 수익인증 반대, 투자철학 학습에 집중 주장
  3. 투자 학습/장기투자 다짐 (~100건)

  서비스 피드백:
  - 자동결제 환불 요청 3건
  - 오프라인 강의 신청 방법 문의 8건

  → 자세히: [대시보드 링크]
```

```
김팀장: 수익인증 논란 원문 보여줘

VOC Bot:
  🔍 "수익인증" 관련 원문 (상위 5건)

  1. [부정] _"'증권사 공식 주문 체결 내역서'나 '세무서 발행
     소득금액증명원'을 단 한 번이라도 투명하게 공개해 주십시오..."_

  2. [긍정/옹호] _"학우분들, 두환쌤의 수익인증이 왜 필요합니까?
     인신 공격으로 비춰집니다..."_

  3. [부정] _"자칭 수천억을 굴린다는 분의 행보치고는 좀..."_

  ... 외 77건 → [전체 보기]
```

### 경로 C: 주간 리포트 (자동 생성)

매주 월요일 09:00, Slack #voc-주간리포트 에 자동 게시.
기존 example.md와 동일한 형식이되, **분류 파이프라인 기반으로 자동 생성**.

```
📝 주간 VOC 리포트 (2026-01-05 주차) 가 생성되었습니다.

📊 핵심 요약
  - 전체 편지 923건(+27.5%), 게시글 1,145건(+166%)
  - 🔴 박두환: 부정 급증 (+13%p) — 수익인증 논란
  - 🟡 미과장: 부정 소폭 상승 — 커뮤니티 분위기 우려
  - 🟢 서재형: 고인 애도 중심, 긍정적 분위기

  → 전체 리포트: [마크다운 링크]
  → 대시보드: [Streamlit 링크]
```

## 시나리오 3: 콘텐츠팀 박PD — "콘텐츠 반응 전반적으로 어때?"

### AI Agent (채팅 인터페이스)

```
박PD: 이번 주 콘텐츠 반응 요약해줘

Agent:
  📊 콘텐츠 반응 요약 (2026-01-05 주차, 총 312건)

  감성 분포: 긍정 245건(79%) / 중립 42건(13%) / 부정 25건(8%)

  ✅ 긍정 Top 주제:
  1. 마스터 응원/격려 (89건) — 서재형 형님 상 위로
  2. 강의/수업 피드백 (67건) — 투자전략 강의 만족
  3. 콘텐츠 품질/깊이 (45건) — 연간전망 리포트 호평

  ⚠️ 부정 Top 주제:
  1. 콘텐츠 품질/깊이 (9건) — "반복적 브리핑" (미과장 5건)
  2. 마스터 소통/태도 (7건) — "질문 미답변" (홍춘욱 4건)
  3. 강의/수업 피드백 (5건) — "온라인 수업 성의 없음" (김영익 3건)
  4. 콘텐츠 접근 문제 (4건) — "16종목 어디서 보는지 모름" (이정윤)

  📌 액션 포인트:
  - 미과장: 브리핑 콘텐츠 다양화 검토
  - 홍춘욱: 질문 답변 프로세스 확인
  - 김영익: 온라인/오프라인 수업 품질 격차 해소
  - 이정윤: 콘텐츠 위치 안내 개선 (온보딩)
```

## 시나리오 4: 경영진 주간 브리핑

### 자동 생성 리포트 핵심 요약 (매주 월요일 Slack 게시)

```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

| 구분        | 이번 주 | 전주  | 증감       |
|------------|--------|------|-----------|
| 편지        |    923 |  724 | +199 (+28%)|
| 게시글      |  1,145 |  430 | +715(+166%)|
| 채널톡 CS   |    47  |   38 | +9 (+24%) |

🔴 위험 신호
  박두환: 부정 28%(+13%p) — 수익인증 투명성 논란 확산
  → 운영팀 모니터링 중, 커뮤니티 가이드라인 검토 필요

🟡 주의
  미과장: 부정 22%(+8%p) — 커뮤니티 내 소수 회원 논쟁
  결제 이슈: 채널톡+편지 동시 급증 — 자동결제 갱신 점검 중

🟢 긍정 시그널
  서재형: 게시글 +699건 — 마스터 가족 상 애도 커뮤니티 결속
  오종태: 3기 오프라인 전우회 호평 — 오프라인 이벤트 효과 입증

📌 부서별 액션
  운영팀: 박두환 수익인증 논란 대응 방안 수립
```
